Support

:: Commercial support ::


If you have a commercial license, you can send an E-mail to support@javadocking.com.
Make sure you add the following information:

  • Your license number.
  • The Sanaware Java Docking version you are using.
  • The Java version you are using.
  • The OS and platform you are using.

:: Services ::

We offer consulting, design, and development services for Java Swing user interface creation. We assist you also in using the Sanaware Java Docking framework in your applications.

If you want to purchase Services , you can contact sales@javadocking.com.
